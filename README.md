## ⚡ Setup

```bash
# Instalar dependências
mix setup

# Iniciar servidor
mix phx.server

# Ou dentro do IEx
iex -S mix phx.server
```

Acesse [`localhost:4000`](http://localhost:4000) no navegador.

## 🚀 Deploy

A aplicação roda na [Fly.io](https://fly.io) na região `gru` (São Paulo).

```bash
fly deploy
```
